# Сергей Гессень: культурная диффузия

**Sankey-диаграмма**: поток публикаций между языками и странами.

## Составители
Azamat Safarov, Oleg Osovsky

## Описание
Интерактивная визуализация с тремя режимами:
1. **Язык → Страна** — из каких языков в какие страны текли публикации
2. **Страна → Язык** — какие языки использовались в каждой стране
3. **Генеалогия идей** — эволюция философских тем от источников к наследию

## Особенности
- Аннотация по годам (1908–1975)
- Интерактивные подсказки при наведении
- Клик на узел — подсветка связей
- Анимация появления полос
- Адаптивный дизайн

## Данные
`gessen_sankey_data.json` — 3 вида связей, 27+ узлов, прямая реальная информация.

## Process Notes
- D3.js Sankey plugin через unpkg
- SVG viewBox для адаптивности
- CSS-переменные для ночной темы
- Линейные градиенты на полосах для плавных переходов цветов
- Аварийная обработка: если фильтр по году обнуляет связи, показываются данные за первый доступный год
